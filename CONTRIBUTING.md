# Contributing

Open an issue before substantial changes. Do not submit real patient, justice, or identifiable data, model weights trained on such data, credentials, or screenshots of secure systems. Contributions should include tests, document assumptions, preserve the research-only warning, and follow the ethical red lines in `docs/GOVERNANCE.md`.

Protocol changes that could affect results must be versioned and labelled as pre- or post-outcome-access. Use conventional commits where practical.
